# 📝 BlogSpace – Full Stack Blogging Platform

🔗 Live Demo: https://blogplatformmm.netlify.app

## About the Project

BlogSpace is a full-stack blogging platform that allows users to create, edit, and share blog posts. Users can register, log in securely, comment on posts, and manage their profiles through a clean and responsive interface.

## Features

- User Registration & Login
- JWT Authentication
- Create, Edit & Delete Posts
- Comment System
- User Profile Page
- Responsive UI
- MongoDB Atlas Integration

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcrypt.js

## Deployment

- Frontend: Netlify
- Backend: Render
- Database: MongoDB Atlas
